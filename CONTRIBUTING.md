# How to contribute

## How to get started

Install a virtualenv enviroment to test the code before publishing it to github.

Example in macOs:
```bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

Example in Windows:
```bash
virtualenv venv
venv/Scripts/activate
pip install -r requirements.txt
```

Test the library before uploading changes
```bash
python -m unittest
```
